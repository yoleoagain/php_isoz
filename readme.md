# Зачетная работа
## Инсталяция
```
docker-compose -up --build
```
## Задание
```
Вам необходимо разработать небольшой портал для туристической фирмы "ТурЭкспресс" с использованием HTML и PHP. Портал должен включать следующие основные функции:
Регистрация пользователей:
Реализуйте форму регистрации новых пользователей.Храните данные о пользователях (имя, email, пароль) в базе данных MySQL.
Авторизация пользователей:Создайте страницу входа для уже зарегистрированных пользователей.
Обеспечьте защиту паролей при хранении и передаче (например, хеширование).Страница выбора туров:
Отобразите список доступных туров с кратким описанием каждого.Добавьте фильтры для поиска туров по разным критериям (цена, страна, продолжительность и т.д.).
Личный кабинет:После авторизации пользователь может попасть в свой личный кабинет.
В личном кабинете должна быть информация о заказанных турах (если такие имеются), а также возможность заказать новый тур.База данных:
Используйте базу данных MySQL для хранения информации о пользователях, турах и заказах.Настройте таблицы для хранения всех необходимых данных.
Общий дизайн и функциональность:Портал должен выглядеть эстетично и современно.
Все формы и страницы должны корректно работать без ошибок.Требования к выполнению
Код должен быть чистым и хорошо структурированным.Используйте комментарии там, где это необходимо для пояснения логики работы программы.
Структура базы данных должна быть оптимальной и соответствовать требованиям задачи.Интерфейс портала должен быть интуитивным и удобным для пользователя.
Примерное распределение балловРегистрация и авторизация – 20%
Страница выбора туров – 30%Личный кабинет – 25%
База данных – 15%Общий дизайн и функциональность – 10%
```
